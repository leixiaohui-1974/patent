# PF1-5：一种基于物理信息神经网络的水网数字孪生快速构建方法

## （一）技术领域

本发明属于水利工程数字孪生与人工智能交叉技术领域，具体涉及一种面向长距离输水明渠及区域水网的数字孪生快速构建方法，尤其涉及将Saint-Venant机理约束嵌入神经网络训练的物理信息神经网络（PINN）建模、少样本参数校准与跨工程迁移复用方法。本发明对应水系统控制论（CHS）八原理中的降阶原理（P3）和自适应原理（P7），为水网自主运行等级（WNAL）L2至L3的跃升提供模型底座支撑。

## （二）背景技术

现有水网数字孪生建模普遍采用“物理建模+人工校参”路线。该路线在单工程深度建设中可获得较高精度，但在跨区域复制时暴露出建模周期长、对专家经验依赖强、参数迁移困难等问题。以长距离明渠为例，若按传统方法逐渠池建立Saint-Venant方程模型、率定糙率和边界条件，常需数周至数月，难以支撑HydroOS快速部署需求。

目前公开技术主要包括：

1. 基于一维/二维数值水动力模型的高精度仿真方法，机理可信，但在线计算与参数率定成本高；
2. 纯数据驱动神经网络方法，训练快但缺乏守恒约束，外推工况下易失真；
3. 机理-数据混合方法，虽然兼顾一定精度与效率，但普遍缺少“少样本+迁移学习+工程化自动校准”一体化流程。

与本发明最接近的现有研究为PINN在流体力学中的应用、以及水文水动力混合建模工作。现有方法虽证明了“物理约束嵌入神经网络”的可行性，但在水网工程场景仍存在不足：一是对边界工况切换、闸泵执行器约束处理不足；二是缺少面向渠池网络拓扑的分层训练组织；三是跨工程迁移多依赖人工经验，不具备标准化快速交付流程。

因此，亟需一种面向水网工程的PINN数字孪生快速构建方法，在保持物理一致性的同时降低数据依赖和部署成本。

## （三）发明内容

### 1. 技术问题

本发明要解决的技术问题是：传统数字孪生构建周期长、率定成本高、跨工程复用差，纯数据驱动方法物理一致性不足，难以在少样本条件下稳定泛化。

### 2. 技术方案

本发明提供一种基于物理信息神经网络的水网数字孪生快速构建方法，包括如下步骤：

**步骤S1：构建水网拓扑与机理约束集合。**

根据渠道、闸门、泵站和分水口构建有向图拓扑，建立节点连续方程与边界控制方程；将Saint-Venant方程组、质量守恒和边界流量约束写入物理残差集合。

**步骤S2：建立PINN主网络与多任务输出。**

构建输入为时空坐标与边界工况、输出为水位 $H(x,t)$ 与流量 $Q(x,t)$ 的多层神经网络；采用自动微分计算 $\partial H/\partial t$、$\partial Q/\partial x$ 等项。

**步骤S3：定义联合损失函数。**

\[
\mathcal{L}=\omega_d\mathcal{L}_{data}+\omega_p\mathcal{L}_{phys}+\omega_b\mathcal{L}_{bc}+\omega_r\mathcal{L}_{reg}
\]

其中，$\mathcal{L}_{data}$ 为观测点误差，$\mathcal{L}_{phys}$ 为Saint-Venant残差，$\mathcal{L}_{bc}$ 为边界条件误差，$\mathcal{L}_{reg}$ 为正则项；采用动态权重平衡策略自适应更新 $\omega_d,\omega_p,\omega_b$。

**步骤S4：执行少样本校准。**

在每个渠池仅配置少量监测数据（闸前/闸后水位、关键断面流量）条件下，通过分阶段训练实现参数识别：先冻结深层特征提取层，再微调输出层和边界嵌入层，降低小样本过拟合风险。

**步骤S5：跨工程迁移学习。**

以源工程训练好的PINN作为预训练模型，针对目标工程通过参数映射矩阵 $\mathbf{M}_{geo}$ 将渠池几何尺度、糙率先验和调度工况映射到新域，仅进行低学习率微调。

**步骤S6：在线偏差检测与闭环更新。**

部署后持续计算孪生-实测偏差指数：

\[
\Delta_k=\alpha_H\|H_k^{meas}-H_k^{twin}\|+\alpha_Q\|Q_k^{meas}-Q_k^{twin}\|
\]

当 $\Delta_k>\theta$ 连续超阈触发在线增量学习，更新模型参数并同步至HydroOS模型层。

### 3. 有益效果

1. 将物理约束嵌入学习过程，显著降低外推失真风险；
2. 在稀疏监测条件下保持较高精度，降低传感器依赖；
3. 通过迁移学习缩短新工程建模周期；
4. 支持在线增量更新，增强数字孪生长期可用性；
5. 与HydroOS接口兼容，可快速接入控制与评估流程。

## （五）具体实施方式

下面结合南水北调中线工程和胶东调水工程实际场景，对本发明的技术方案进行详细说明。

### 实施例1：南水北调中线典型渠段PINN数字孪生构建

**1. 工程概况与数据配置**

选取南水北调中线总干渠安阳—邯郸段（桩号约580km至660km）共80km渠段作为孪生建模对象。该段包含6个渠池，由7座节制闸（G1~G7）控制，渠池长度分布为12km、15km、18km、10km、13km、12km。渠道梯形断面，底宽b = 16.0 m，边坡系数m = 2.5，纵底坡S₀ = 1/25000，Manning糙率n = 0.015。设计流量Q₀ = 120 m³/s，设计水深y₀ = 5.0 m。

SCADA系统采样周期T_s = 60 s，监测点配置如下：

| 监测类型 | 数量 | 位置 | 精度 |
|----------|------|------|------|
| 闸前水位 | 7 | 各闸门上游50m | ±0.5 cm |
| 闸后水位 | 7 | 各闸门下游50m | ±0.5 cm |
| 闸门开度 | 7 | 各闸门 | ±0.1 cm |
| 渠池中部水位 | 6 | 各渠池中点 | ±1.0 cm |
| 分水口流量 | 3 | 桩号590/620/645km | ±2% |

总计30个监测量，占全渠段计算断面数（Δx=200m时共400个断面）的7.5%。训练数据选取2024年3—10月共8个月的历史运行数据，包含正常输水（Q = 80~130 m³/s）、流量调节过渡、分水口启停等多种工况。

**2. 水网拓扑与机理约束集合构建（步骤S1）**

根据渠道节制闸拓扑建立有向图G = (V, E)，其中节点集V = {G1, ..., G7, D1, D2, D3}包含7座闸门和3个分水口，边集E表示渠池连接关系。

物理残差集合包括：

（a）Saint-Venant方程残差——对每个空间采样点(x_k, t_j)：

$r_{cont}(x_k, t_j) = \frac{\partial A}{\partial t} + \frac{\partial Q}{\partial x}\bigg|_{(x_k, t_j)}$

$r_{mom}(x_k, t_j) = \frac{\partial Q}{\partial t} + \frac{\partial (Q^2/A)}{\partial x} + gA\frac{\partial y}{\partial x} - gA(S_0 - S_f)\bigg|_{(x_k, t_j)}$

其中偏导数通过PINN的自动微分计算，无需有限差分离散。

（b）节点质量守恒残差——在每个闸门节点i：

$r_{node,i} = Q_{i,in} - Q_{i,out} - Q_{i,take}$

（c）边界流量约束残差——闸门过流公式：

$r_{gate,i} = Q_i - C_d \cdot b \cdot e_i \cdot \sqrt{2g(H_{up,i} - H_{dn,i})}$

**3. PINN网络结构与联合损失函数（步骤S2-S3）**

网络结构采用8层全连接结构，每层128个神经元，激活函数为tanh。网络输入维度为d_in = 2 + 7 + 3 + 1 = 13，分别为：时空坐标(x, t)共2维、7个闸门开度、3个分水口取水流量、1个气温参数。网络输出维度为d_out = 2，分别为水位H(x,t)和流量Q(x,t)。

联合损失函数各项的具体计算方法：

$\mathcal{L}_{data} = \frac{1}{N_d}\sum_{i=1}^{N_d}\left[\left(H^{pred}_i - H^{meas}_i\right)^2 + \left(Q^{pred}_i - Q^{meas}_i\right)^2\right]$

其中$N_d$为监测点数据总量。以60秒采样周期、8个月数据计算，$N_d = 30 \times 8 \times 30 \times 24 \times 60 = 6.22 \times 10^6$个数据点（训练时按批次随机采样）。

$\mathcal{L}_{phys} = \frac{1}{N_p}\sum_{j=1}^{N_p}\left[r_{cont}^2(x_j, t_j) + r_{mom}^2(x_j, t_j)\right]$

其中$N_p$为物理配点数。在时空域内采用拉丁超立方采样，每批次取$N_p = 10000$个配点。

$\mathcal{L}_{bc} = \frac{1}{N_b}\sum_{k=1}^{N_b}\left[r_{node,k}^2 + r_{gate,k}^2\right]$

其中$N_b$为边界采样点数，每批次取$N_b = 2000$。

$\mathcal{L}_{reg} = \lambda_{L2}\sum_l \|\mathbf{W}_l\|_F^2$

正则化系数$\lambda_{L2} = 10^{-5}$。

动态权重更新策略采用GradNorm方法：每100个训练步计算各损失项的梯度范数比值，将权重向梯度失衡方向调整。初始权重$\omega_d=1.0, \omega_p=2.5, \omega_b=1.2, \omega_r=0.01$。

**4. 分阶段训练过程（步骤S4）**

训练分三阶段执行：

阶段一（机理预训练，8000次迭代）：仅优化$\mathcal{L}_{phys}+\mathcal{L}_{bc}$，学习率$lr=1\times10^{-3}$，优化器Adam。该阶段使网络首先学会满足物理约束，即使无实测数据也能产生合理的水动力场分布。训练结束时$\mathcal{L}_{phys}$降至$3.2\times10^{-4}$。

阶段二（数据融合，12000次迭代）：加入$\mathcal{L}_{data}$，四项损失联合优化。学习率从$1\times10^{-3}$按余弦退火降至$2\times10^{-5}$。训练结束时各损失项收敛值：$\mathcal{L}_{data}=1.8\times10^{-5}$，$\mathcal{L}_{phys}=2.1\times10^{-4}$，$\mathcal{L}_{bc}=8.5\times10^{-6}$。

阶段三（轻量微调，3000次迭代）：冻结前4层（特征提取层），仅训练后4层和输出层。学习率$lr=5\times10^{-5}$。该阶段的目的是在不破坏已学到的物理特征的前提下精细拟合监测点数据。

训练硬件：NVIDIA A100 GPU（40GB显存），训练总时长约4.5小时。

**5. 孪生精度验证**

以2024年11—12月两个月数据作为独立测试集（未参与训练），验证结果如下：

| 指标 | 水位H | 流量Q |
|------|-------|-------|
| RMSE | 0.018 m | 1.42 m³/s |
| MAE | 0.013 m | 0.98 m³/s |
| MAPE | 0.26% | 2.9% |
| R² | 0.997 | 0.994 |
| 最大误差 | 0.058 m | 5.3 m³/s |

对比分析：相同渠段采用传统Preissmann格式Saint-Venant数值模型（经过4周人工率定后），水位RMSE为0.012 m，流量MAPE为2.1%。本发明PINN方法精度略低于精细率定的传统模型，但建模周期从4周缩短至4.5小时（训练时间），且无需专家手动率定糙率等参数。

上述结果体现了物理信息神经网络方法的独特优势。传统水动力建模方法需要专业人员逐段率定糙率、分水系数等参数，不同率定人员的结果往往存在显著差异，且率定周期通常需要数周时间。而本发明将Saint-Venant方程作为硬约束嵌入网络训练过程，使模型自动学习到符合水力学规律的参数映射关系。从测试结果看，水位预测的决定系数高达0.997，说明模型几乎完全捕获了水位的时空变化特征，包括闸门调节引起的回水效应和分水口启停导致的局部水位波动。流量预测最大误差出现在分水口取水量快速变化的时段，这与流量监测设备本身的响应滞后特性有关。

物理配点的采样策略对模型精度有重要影响。拉丁超立方采样能够在时空域内均匀分布配点，避免了均匀网格采样可能导致的局部欠约束问题。每批次一万个配点的规模经实验验证，能够在计算效率和约束覆盖率之间取得良好平衡——过少的配点会导致网络在无监测数据的渠段产生违反物理规律的异常预测，过多则显著增加训练耗时。分阶段训练策略中，第一阶段的机理预训练尤为关键，它使网络首先建立对水流连续性和动量守恒的基本认知，后续引入实测数据时仅需微调而非从零学习物理规律。

### 实施例2：胶东调水工程迁移复用

**1. 目标工程概况**

胶东调水工程引黄济青段，管道总长约120km，沿线设有8座泵站、5座调压井。与南水北调中线的明渠系统不同，胶东调水为有压管道输水系统，但水力方程同属Saint-Venant方程族（管道可视为Family α积分型传递函数的特殊情况，详见T2a第5章管道弹性容量模型）。

目标工程仅提供14天（2025年1月1—14日）的历史运行数据用于迁移校准，包括8个泵站出口压力、5个调压井水位和干线3个断面流量，合计16个监测点。

**2. 几何映射与迁移训练（步骤S5）**

几何映射矩阵$\mathbf{M}_{geo}$将源工程（南水北调中线明渠）的特征空间映射到目标工程（胶东管道）的特征空间：

$\mathbf{M}_{geo} = \begin{bmatrix} L_{target}/L_{source} & 0 & 0 \\ 0 & A_{target}/A_{source} & 0 \\ 0 & 0 & n_{target}/n_{source} \end{bmatrix}$

其中L为渠段/管段长度比、A为过水面积比、n为糙率比。对于明渠到管道的迁移，还需附加管道弹性项的约束嵌入。

迁移训练策略：冻结前6层（保留源工程学到的通用水动力特征），仅微调后2层和输出层。边界嵌入层需要重新初始化以适应泵站边界条件（替代闸门边界条件）。学习率设为$5\times10^{-5}$，迭代2500次，训练时间约35分钟。

**3. 迁移效果**

| 对比项 | 传统建模 | PINN迁移 |
|--------|---------|----------|
| 建模周期 | 6~8周 | 5天（含数据整理） |
| 水位RMSE | 0.028 m（经率定） | 0.022 m |
| 流量MAPE | 4.1% | 3.3% |
| 所需历史数据量 | ≥3个月 | 14天 |
| 是否需要专家率定 | 是 | 否 |

PINN迁移模型在仅14天数据条件下即可达到甚至优于传统方法的精度，主要得益于预训练模型已掌握了水动力学的通用物理规律。迁移学习的核心在于源工程与目标工程之间存在共享的物理不变量——连续性方程和动量方程的基本形式不因工程几何而改变，变化的仅是几何参数和边界条件类型。冻结前六层保留了网络对流体动力学基本模式的编码能力，包括波的传播、衰减和反射等通用特征。仅微调后两层使网络快速适应管道系统特有的水锤波速和弹性效应，这种策略将所需训练数据量从完整训练的三个月压缩至两周，从根本上改变了数字孪生部署的工程经济性。对于全国数百处待建的智慧水网工程而言，该迁移框架可将单个工程的数字孪生建设周期从数月缩短至一周以内，极大加速了水网操作系统的规模化推广部署。

### 实施例3：在线偏差自校准（步骤S6）

**1. 偏差监测配置**

在南水北调中线实施例部署后，系统每T_w = 5分钟计算一次孪生-实测偏差指数：

$\Delta_k = \alpha_H \cdot \frac{1}{N_H}\sum_{i=1}^{N_H}|H_i^{meas}(k) - H_i^{twin}(k)| + \alpha_Q \cdot \frac{1}{N_Q}\sum_{j=1}^{N_Q}\frac{|Q_j^{meas}(k) - Q_j^{twin}(k)|}{Q_j^{meas}(k)}$

其中$\alpha_H = 50$（将水位误差量纲归一化到与流量百分比误差同量级），$\alpha_Q = 1$，$N_H = 20$（水位监测点），$N_Q = 10$（流量监测点）。

告警阈值$\theta = 0.06$，连续超阈窗口数$N_{trigger} = 3$（即连续15分钟偏差超限则触发更新）。

**2. 汛期工况切换验证**

在2025年7月汛期调度中，上游来流在2小时内从Q = 95 m³/s增至Q = 140 m³/s（增幅47%），同时3个分水口同步增大取水量。该工况变化超出原训练数据覆盖范围。

偏差指数响应过程：
- t = 0 min：来流开始增加，$\Delta_0 = 0.032$（正常范围）
- t = 30 min：偏差快速上升，$\Delta_6 = 0.071$（首次超阈）
- t = 40 min：连续3个窗口超阈，触发增量学习
- t = 60 min：增量学习启动，学习窗口覆盖t = -120min至t = 0的数据（2小时滚动窗口），学习率$lr = 1\times10^{-5}$，仅更新后2层参数
- t = 80 min：增量学习完成（约400次迭代），$\Delta_{16} = 0.041$
- t = 100 min：偏差回落至$\Delta_{20} = 0.028$，恢复正常区间

整个自校准过程中，控制层使用孪生预测进行前馈补偿时，水位最大偏差从无自校准时的0.12 m降低至0.083 m（降低31%）。

**3. 长期运行统计**

连续部署6个月（2025年4—9月）的运行统计：

| 指标 | 数值 |
|------|------|
| 增量学习触发次数 | 7次 |
| 平均触发间隔 | 25.7天 |
| 平均自校准耗时 | 18.5分钟 |
| 自校准后偏差降低率 | 42%±11% |
| 模型可用率（偏差指数<阈值的时间占比） | 99.2% |
| 相比传统模型的人工维护工时节省 | ~80% |

上述结果表明，本发明的PINN数字孪生方法在保持物理一致性的前提下，可大幅缩短建模周期、降低数据依赖和运维成本，为HydroOS水网操作系统的快速部署和长期可靠运行提供模型底座支撑。该方法支撑了WNAL L2（辅助运行）到L3（有条件自主运行）的能力跃升——通过可靠的数字孪生实时预测，使自主控制系统能够在已验证的运行设计域（ODD）内独立决策。

## （七）摘要

本发明公开了一种基于物理信息神经网络的水网数字孪生快速构建方法。该方法将Saint-Venant方程、质量守恒和边界约束嵌入神经网络训练，通过“数据项+物理项+边界项”联合损失实现机理一致建模；在稀疏监测条件下采用分阶段少样本校准，并通过几何参数映射实现跨工程迁移复用；部署后基于孪生-实测偏差指数触发在线增量学习，实现持续自校准。实施结果表明，本方法可显著缩短数字孪生建模周期并提高跨工况预测稳定性，适用于HydroOS快速部署场景。
